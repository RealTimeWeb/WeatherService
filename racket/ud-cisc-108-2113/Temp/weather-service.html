1   Introduction

This teachpack let’s you get the weather and forecast for a given address. You can use it online, with live data:

  ( require "weather-service-online.rkt" )

Why would you use it offline? First, you don’t need an internet connection. Second, you don’t have to worry about your tests changing, since local data never changes (even though weather should change daily!).

or offline with local data:
  ( require "weather-service-offline.rkt" )

2   Functions

procedure

( get-weather   address )  →   ( or/c   string?   report? )
   address  :   string?
Gets a report on the current weather, the forecast, and more detailed information about the given address, UNLESS there was an error, in which case the error is returned as a string.

  • address — An address in the United States, such as "Seattle, WA" or "101 Smith Hall, Newark, DE 19716".

When using the service offline, you should only use the following addresses:

  • "Newark, DE"

  • "Seattle, WA"

  • "Orlando, FL"

  • "Washington DC"

  • "Los Angeles, CA"

  • "Chicago, IL"

  • "Alaska"

There are several reasons why get-weather might fail:

  • The address might be outside of the United States.

  • There might have been no results found for the address (try it in Google Maps to make sure it’s a real address)

  • If you’re using the offline version, the address may not be available offline. You can try it with the online version.

3   Structs

struct

( struct   report   ( current tonight tomorrow day-after location )
    #:extra-constructor-name   make-report )
   current  :   weather?
   tonight  :   forecast?
   tomorrow  :   forecast?
   day-after  :   forecast?
   location  :   location?
A container for the weather, forecasts, and location information.

  • current — The current weather for this address.

  • tonight — The forecast for tonight.

  • tomorrow — The forecast for tomorrow.

  • day-after — The forecast for the day after tomorrow.

  • location — More detailed location information on this address.

struct

( struct   weather   ( temperature
        dewpoint
        humidity
        visibility
        pressure
        wind
        description
        image-url )
    #:extra-constructor-name   make-weather )
   temperature  :   number?
   dewpoint  :   number?
   humidity  :   number?
   visibility  :   number?
   pressure  :   number?
   wind  :   wind?
   description  :   string?
   image-url  :   string?
A structured representation the current weather.

  • temp — The current temperature (in Fahrenheit).

  • dewpoint — The current dewpoint temperature (in Fahrenheit).

  • humidity — The current relative humidity (as a percentage).

  • visibility — How far you can see (in miles).

  • pressure — The barometric pressure (in inches).

  • wind — The current wind pattern.

  • description — A human-readable description of the current weather.

  • image-url — A url pointing to a picture that describes the weather. Use "bitmap/url" to convert it to an image.

struct

( struct   wind   ( speed direction chill )
    #:extra-constructor-name   make-wind )
   speed  :   number?
   direction  :   number?
   chill  :   number?
A structured representation of wind.

  • speed — The current wind speed (in miles-per-hour).

  • direction — The current wind direction (in degrees).

  • chill — The perceived temperature (in Fahrenheit) caused by the wind.

struct

( struct   forecast   ( temperature-label
        temperature
        probability-of-precipitation
        image-url
        description
        long-description )
    #:extra-constructor-name   make-forecast )
   temperature-label  :   string?
   temperature  :   number?
   probability-of-precipitation  :   number?
   image-url  :   string?
   description  :   string?
   long-description  :   string?
A prediction for future weather.

  • temperature-label — Either "High" or "Low", depending on whether or not the predicted temperature is a daily high or a daily low.

  • temperature — The predicted temperature for this period (in Fahrenheit).

  • probability-of-precipitation — The probability of precipitation for this period (as a percentage).

  • image-url — A url pointing to a picture that describes the predicted weather for this period. Use "bitmap/url" to convert it to an image.

  • description — A human-readable description of the predicted weather for this period.

  • long-description — A more-detailed, human-readable description of the predicted weather.

struct

( struct   location   ( latitude longitude elavation name )
    #:extra-constructor-name   make-location )
   latitude  :   number?
   longitude  :   number?
   elavation  :   number?
   name  :   string?
A detailed description of a location

  • latitude — The latitude (up-down) of this location.

  • longitude — The longitude (left-right) of this location.

  • elavation — The height above sea-level (in feet).

  • name — The city and state that this location is in.
